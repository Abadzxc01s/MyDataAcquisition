How It Works

        

Setup the DAQ System

Setting up the system close to your plant farm and server. Remember that the system use bluetooth and has a maximum range.

  

Connect your laptop bluetooth to the DAQ System bluetooth

Pairing your laptop bluetooth to the DAQ System bluetooth is necessary since it would the transmission medium of the data acquired of your system going to your server database.

       

Run the application

After pairing the bluetooth you should run the DAQ System application and run it to start acquiring data from your plant farm

    

View the data

After doing the following steps. Your are now ready to view the data you have been acquired and learn how this data affect the growth of your plants in terms of Vapor Pressure Deficit
